# musync
A program that gets your music from Spotify and downloads it from Youtube Music.

# Installation
## Using Pip
```bash
  $ pip install musync
```
## Manual
```bash
  $ git clone https://github.com/vittodevit/musync
  $ cd musync
  $ python setup.py install
```
# Usage
```bash
$ musync [configfile]
```
